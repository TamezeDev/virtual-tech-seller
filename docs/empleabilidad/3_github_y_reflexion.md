## Perfil Profesional en GitHub

Para establecer mi marca personal como desarrollador, he configurado mi perfil de GitHub siguiendo las siguientes prácticas:

## Reflexión Personal

Mirando en retrospectiva este último mes de trabajo en el Proyecto Intermodular, saco conclusiones muy valiosas sobre mi evolución como programador:

* **¿Qué he aprendido?** He aprendido que desarrollar software no es solo escribir líneas de código hasta que no haya errores de compilación. He aprendido a pensar en Arquitectura (separar responsabilidades), a persistir datos de forma segura (SQL y JDBC), a interconectar sistemas (exportación XML/JAXB) y a proteger la aplicación (encriptación e integridad referencial).
* **¿Qué se me ha dado mejor?** La estructuración del Backend y el diseño de la Base de Datos. Me he sentido muy cómodo aplicando Patrones de Diseño (como el patrón Singleton para la sesión, o el Factory para la gestión de usuarios según su rol) y modelando la lógica de las entidades mediante Programación Orientada a Objetos pura.
* **¿Qué me ha costado más?** A medida que el código crece, es más difícil mantener que la lógica de negocio se cumpla con lo establecido desde el principio, ya que distintas partes del programa atacan a tablas de datos que son leídas desde diversos módulos. Interconectar de forma fluida el Backend requirió mucho ensayo y error para evitar que algunos datos se pisaran o se perdieran.
* **¿Qué mejoraría?** Si tuviera más tiempo, me gustaría haber implementado pruebas unitarias automatizadas (JUnit) de servicios y validaciones para asegurar que los cálculos de impuestos y descuentos nunca fallen, así como separar la interfaz gráfica consumiendo una API REST desde mi backend Java, de tal forma que las consultas queden aisladas, brindando una seguridad real tanto para el cliente como para el usuario.
* **¿Qué pasos quiero dar para acercarme al perfil profesional que me interesa?** Mi siguiente paso a corto plazo es dominar el framework **Spring Boot** para transformar aplicaciones monolíticas como esta en verdaderas APIs o Microservicios web. Del mismo modo, quiero adquirir los conocimientos necesarios para poder llevar estos proyectos a Android o iOS y así poder disfrutar las aplicaciones desde la mayor cantidad de dispositivos posibles. Por otro lado, también quiero profundizar en la administración avanzada de bases de datos relacionales y obtener certificaciones en Ciberseguridad para asegurar que mis futuros desarrollos empresariales estén blindados desde el minuto cero.
